# SnapLab

网页助手插件，包含图片预览、缩放、旋转等图片处理相关功能。一个极简且强大的浏览器扩展。

## 功能特点

- **极简设计**：无需复杂配置，开箱即用。
- **全功能预览**：支持滚轮缩放、鼠标拖拽、旋转、水平/垂直翻转。
- **跨浏览器**：支持 Chrome, Firefox, Edge 等主流浏览器。

## 安装与开发

本项目使用 [WXT](https://wxt.dev/) 框架开发。

### 开发
```bash
pnpm install
pnpm dev
```

### 打包
项目使用 [Task](https://taskfile.dev/) 进行任务管理。
```bash
task publish
```
